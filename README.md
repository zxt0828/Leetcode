# LeetCode 错题集 📝

记录刷题过程中做错的题目，复盘思路、总结易错点。

## 目录结构

| 分类 | 说明 |
|------|------|
| Array | 数组 |
| Backtracking | 回溯 |
| Binary_Search | 二分查找 |
| Binary_Search_Tree | 二叉搜索树 |
| Binary_Tree | 二叉树 |
| Breadth_First_Search | 广度优先搜索 |
| Depth_First_Search | 深度优先搜索 |
| Design | 设计题 |
| Dynamic_Programming | 动态规划 |
| Graph | 图 |
| Greedy | 贪心 |

| String | 字符串 |
| Tree | 树 |
| Two_Pointers | 双指针 |

## 模版格式

每道题一个 `.md` 文件，命名格式：`题号_题目名.md`

例如：`875_Koko_Eating_Bananas.md`

详见各文件夹中的示例。
