# 18. 4Sum

**难度:** Medium

**标签:** Two Pointers

## 题目链接

[LeetCode 18](https://leetcode.com/problems/4sum/)

## 代码

```java
class Solution {
    public List<List<Integer>> fourSum(int[] nums, int target) {
        List<List<Integer>> res = new ArrayList<>();
        if (nums.length < 4) return res;
        Arrays.sort(nums);

        for (int i = 0; i < nums.length - 3; i++) {
            if (i > 0 && nums[i] == nums[i - 1]) continue;  // 第一层去重

            for (int j = i + 1; j < nums.length - 2; j++) {
                if (j > i + 1 && nums[j] == nums[j - 1]) continue;  // 第二层去重

                int left = j + 1;
                int right = nums.length - 1;

                while (left < right) {
                    long sum = (long) nums[i] + nums[j] + nums[left] + nums[right];
                    if (sum == target) {
                        res.add(Arrays.asList(nums[i], nums[j], nums[left], nums[right]));
                        // 内层去重：跳过重复的 left 和 right
                        while (left < right && nums[left] == nums[left + 1]) left++;
                        while (left < right && nums[right] == nums[right - 1]) right--;
                        left++;
                        right--;
                    } else if (sum < target) {
                        left++;
                    } else {
                        right--;
                    }
                }
            }
        }
        return res;
    }
}
```

## 解题心得

- 本质就是 3Sum 外面再套一层 for 循环，固定前两个数，内层双指针找后两个数
- **每一层都需要去重**：外层两个 for 循环各自去重（`continue` 跳过），内层双指针找到解后也要去重（`while` 跳过）
- `sum` 要用 `long`，四个 int 相加可能溢出

## 易错点

### 1. 外层去重不能用 while，要用 if + continue

```java
// 错误：用 while 去重，会跳过合法元素
while (i > 0 && nums[i] == nums[i - 1]) i++;

// 正确：用 if + continue，只跳过当前这一轮
if (i > 0 && nums[i] == nums[i - 1]) continue;
```

外层是 for 循环，用 `continue` 跳到下一轮。用 `while` 会在 for 的 `i++` 之外额外多跳，可能漏掉合法组合。

### 2. 内层去重必须在找到解之后

内层的 `while (nums[left] == nums[left + 1]) left++` 只在 `sum == target` 之后执行，不是每次循环都跑。

### 3. 里外层都需要去重

- 第一层 `i`：`if (i > 0 && nums[i] == nums[i-1]) continue`
- 第二层 `j`：`if (j > i+1 && nums[j] == nums[j-1]) continue`
- 内层 `left/right`：找到解后 while 跳过重复值

三层去重缺一不可，否则结果会有重复组合。

## 犯过的错

1. 用一个 while 循环管两个指针代替两层 for 循环，结构完全不对
2. 去重逻辑放在 while 循环外面，导致没有正确去重
3. `new ArrayList<>(a, b, c, d)` 语法不存在，要用 `Arrays.asList(a, b, c, d)`
